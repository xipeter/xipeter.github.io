# Content Previewer

A free online tool to preview and validate Markdown, JSON, and XML content. Compare differences between two versions with the built-in diff viewer. Perfect for developers, technical writers, and anyone working with structured content.

## What You Can Do

### Markdown Preview

Preview Markdown with syntax highlighting for code blocks. See how headings, lists, links, and formatting will render in real-time.

### JSON Preview

Validate and format JSON with pretty printing. Quickly spot syntax errors and see the structure of your data.

### XML Preview

Parse and validate XML documents. Check well-formedness and view the hierarchical structure.

### Diff Comparison

Compare two versions of text side-by-side. Highlight additions, deletions, and changes with color-coded diffs.

## How to Use

1. Select a format from the sidebar (Markdown, JSON, XML, or Diff)
2. Enter or paste your content in the editor
3. Click "Preview" to see how your content will render
4. Use "Copy" to copy the content to your clipboard
5. Use "Clear" to reset and start fresh

## When to Use This Tool

- **Writing documentation** - Preview Markdown before committing to your docs
- **Debugging API responses** - Validate JSON from API calls
- **Working with config files** - Check XML or JSON configuration
- **Code review** - Compare two versions to see what changed
- **Learning** - See how Markdown or JSON looks when rendered
