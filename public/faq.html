Frequently Asked Questions

What is Content Previewer?

Content Previewer is a free online tool that helps you preview and validate Markdown, JSON, and XML content. It also includes a diff viewer to compare two versions of text side-by-side.

Is this tool free to use?

Yes, Content Previewer is completely free. No registration required, no hidden fees.

Does this tool store my data?

No. All processing happens in your browser. Your content is never sent to our servers or stored anywhere. When you refresh the page, your content is cleared.

What is Markdown?

Markdown is a lightweight markup language that allows you to format text using simple syntax. It's commonly used for documentation, readme files, and blog posts. The preview shows you how your Markdown will render when converted to HTML.

How do I validate JSON?

Paste your JSON content into the editor and select JSON format. The tool will automatically check if your JSON is valid. If there are syntax errors, it will show you where the problem is.

What is the Diff viewer used for?

The diff viewer lets you compare two versions of text to see what changed. It's useful for reviewing code changes, tracking document edits, or finding differences between configuration files.

Can I copy the previewed content?

Yes. In preview mode, click the "Copy" button to copy your content to the clipboard. A "Copied!" confirmation will appear briefly.

What languages are supported?

The tool supports 6 languages: English, Chinese (中文), Spanish (Español), Japanese (日本語), German (Deutsch), and French (Français). Switch languages using the language icon in the top right corner.

Does this tool support syntax highlighting?

Yes. Code blocks in Markdown are highlighted using Prism.js with the oneLight theme. JSON and XML also have basic formatting applied.

How do I report issues or suggest features?

You can contact us through the Contact page. We welcome feedback and will try to address issues as quickly as possible.

Is my privacy protected?

Yes. Please read our Privacy Policy for details on how we handle your data. We don't collect personal information unless you contact us directly.
