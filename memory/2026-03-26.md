# 2026-03-26 - TechHub Bilingual Support

## Goal
Implement full bilingual support (EN/ZH) for xipeter.github.io with:
- Locale-prefixed routes (/en.html, /zh.html)
- Clean language switcher
- All pages localized
- Multilingual SEO

## Accomplishments

### Translation System ✅
- Created `src/i18n/en.json` and `src/i18n/zh.json` with all UI strings
- Created `src/i18n/translations.ts` with utilities:
  - `getTranslations()`, `getAlternateLocale()`
  - `getLocalizedPath(basePath, locale)` - constructs localized paths
  - `getPathWithoutLocale(path)` - extracts path without locale prefix
  - `getCurrentLocale(pathname)` - detects current locale

### Pages Localized ✅
- `[lang]/index.astro` - Homepage hub
- `[lang]/articles/index.astro` - Articles hub
- `[lang]/articles/[slug].astro` - Article detail
- `[lang]/tools/index.astro` - Tools hub
- `[lang]/tools/[slug].astro` - Tool detail
- `[lang]/promos/index.astro` - Promos hub
- `[lang]/promos/[slug].astro` - Promo detail
- `[lang]/about.astro`, `/contact.astro`
- `[lang]/privacy-policy.astro`, `/terms.astro`
- `[lang]/editorial-policy.astro`, `/affiliate-disclosure.astro`

### SEO Fixes ✅
- Updated sitemap to include `/en.html` and `/zh.html` routes
- Fixed hreflang tags with correct alternate URLs
- Root `/` stays canonical, locale pages use locale-specific hreflang

### Language Switcher Fix ✅ (Mar 26)
- Language switcher now preserves page context when switching
- `/en/tools/previewer.html` → `/zh/tools/previewer.html`
- `/zh.html` → `/en.html`
- `/en/about.html` → `/zh/about.html`
- Fixed `getPathWithoutLocale()` to handle:
  - Root locale pages (`/en.html`, `/zh.html`) → returns `/`
  - `.html` extension stripping
  - Trailing slash handling

### Cleanup ✅
- Removed orphaned `.lang-switch` and `.lang-btn` CSS from page files
- Cleaned up duplicate language switcher instances

## Files Modified
